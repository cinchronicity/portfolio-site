Project Overview

The Meet App is a serverless, progressive web application (PWA) built with React using a test-driven development (TDD) approach. It integrates with the Google Calendar API to fetch and display upcoming events, allowing users to search by city, view event details, and visualize event data through interactive charts. The app works both online and offline, offering a seamless user experience across all devices using modern advanced React concepts, serverless architecture, and modern web development best practices.

Goal / Purpose

This project demonstrates the ability to build high-quality, scalable applications using serverless architecture, progressive web app principles, and TDD practices. It emphasizes writing maintainable, well-tested code, integrating third-party APIs, and deploying serverless functions through AWS Lambda ‚Äî all while ensuring a responsive, installable, and reliable user experience.

Challenges & Solutions

Challenge: Implementing serverless architecture and managing complex API integrations with proper error handling.

Solution: Built AWS Lambda functions for authentication and API calls, implemented comprehensive error handling and fallback mechanisms.

Challenge: Test coverage for both unit and end-to-end tests in a React PWA environment.

Solution: Implemented multi-layered testing approach using Jest for unit tests, React Testing Library for component integration, and Puppeteer for end-to-end scenarios.

Challenge: Ensuring offline functionality and data synchronization in a PWA.

Solution: Implemented Workbox service workers for caching and background sync, ensuring data availability and synchronization when the network is restored.

Lessons Learned

  • Real-time analytics are crucial for identifying and resolving user experience issues
  • Service workers significantly enhance user experience but require careful cache management
  • Writing tests first improved code quality and reduced debugging time
  • Gained hands-on experience with serverless functions using AWS Lambda
  • Learned how to implement OAuth2 authentication and handle secure API requests
